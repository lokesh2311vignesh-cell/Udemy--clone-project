# My Udemy Clone

A simple Udemy Clone website built using HTML and CSS. This project recreates the basic user interface of the Udemy platform, including a navigation bar, course categories, featured courses, and a clean responsive layout.

## 🚀 Features

- Responsive navigation bar
- Course category section
- Featured courses display
- Clean and modern design
- Mobile-friendly layout
- Built with pure HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📚 Learning Objectives

This project was created to improve my understanding of:

- Semantic HTML structure
- CSS styling and layouts
- Flexbox and Grid concepts
- Responsive web design
- Front-end development fundamentals
